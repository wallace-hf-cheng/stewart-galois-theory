# stewart-galois-theory
My notes and solutions to some of the exercises in Ian Stewart's textbook "Galois Theory".

## Details about the book
- Title: Galois Theory
- Author: Ian Stewart
- Edition: Fifth edition
- Publisher: CRC Press
- Year: 2023
- ISBN: 978-1-003-21394-9
- DOI: 10.1201/9781003213949
- Website: https://www.taylorfrancis.com/books/mono/10.1201/9781003213949/galois-theory-ian-stewart

## About the solutions
- The emphasis of these solutions is on the main mathematical ideas rather than routine calculations. Consequently, straightforward computational details are occasionally omitted. For lengthy or mechanical computations, a computer algebra system (primarily SymPy) may be used. Whenever such computations play a role in a solution, the corresponding code is included.
