\prelem \label{prelem:ex4.2d}

The subset $S = \{ p+q\sqrt{2}+r\imunit+s\imunit\sqrt{2} : p,q,r,s \in \Q \}$ of $\C$ is a subfield.

\pf Let $p_1+q_1\sqrt{2}+r_1\imunit+s_1\imunit\sqrt{2},p_2+q_2\sqrt{2}+r_2\imunit+s_2\imunit\sqrt{2} \in S$, where $p_1,q_1,r_1,s_1,p_2,q_2,r_2,s_2 \in \Q$. Note that \[ (p_1+q_1\sqrt{2}+r_1\imunit+s_1\imunit\sqrt{2}) + (p_2+q_2\sqrt{2}+r_2\imunit+s_2\imunit\sqrt{2}) = (p_1+p_2)+(q_1+q_2)\sqrt{2}+(r_1+r_2)\imunit+(s_1+s_2)\imunit\sqrt{2} \in S. \] Next, we can compute the product using the following Python code.
\begin{minted}{python}
import sympy as sp

p1, q1, r1, s1, p2, q2, r2, s2 = sp.symbols('p1 q1 r1 s1 p2 q2 r2 s2', real=True)
sqrt2 = sp.sqrt(2)
i = sp.I

expression = (p1 + q1*sqrt2 + r1*i + s1*i*sqrt2) * (p2 + q2*sqrt2 + r2*i + s2*i*sqrt2)

print(sp.expand(expression))
\end{minted}
We get
\begin{align*}
& \, (p_1+q_1\sqrt{2}+r_1\imunit+s_1\imunit\sqrt{2})(p_2+q_2\sqrt{2}+r_2\imunit+s_2\imunit\sqrt{2}) \\
=& \, p_1p_2+p_1q_2\sqrt{2}+p_1r_2\imunit+p_1s_2\sqrt{2}\imunit+p_2q_1\sqrt{2}+p_2r_1\imunit+p_2s_1\sqrt{2}\imunit+2q_1q_2 \\
& \, +q_1r_2\sqrt{2}\imunit+2q_1s_2\imunit+q_2r_1\sqrt{2}\imunit+2q_2s_1\imunit-r_1r_2-r_1s_2\sqrt{2}-r_2s_1\sqrt{2}-2s_1s_2 \\
=& \, (p_1p_2+2q_1q_2-r_1r_2-2s_1s_2) + (p_1q_2+p_2q_1-r_1s_2-r_2s_1)\sqrt{2} \\
& \, + (p_1r_2+p_2r_1+2q_1s_2+2q_2s_1)\imunit + (p_1s_2+p_2s_1+q_1r_2+q_2r_1)\sqrt{2}\imunit \\
\in& \, S.
\end{align*}
Also, we have \[ -(p_1+q_1\sqrt{2}+r_1\imunit+s_1\imunit\sqrt{2}) = (-p_1) + (-q_1)\sqrt{2} + (-r_1)\imunit + (-s_1)\imunit\sqrt{2} \in S \] and \[ 1 = 1+0\sqrt{2}+0\imunit+0\imunit\sqrt{2} \in S. \] By the subring test, $S$ is a subring of $\C$. We next find an inverse for any non-zero element in $S$ to show that it is a subfield. Let $p+q\sqrt{2}+r\imunit+s\imunit\sqrt{2} \in S$, where $p,q,r,s \in \Q$. We first define \[ \alpha = (p+q\sqrt{2}+r\imunit+s\imunit\sqrt{2})(p+q\sqrt{2}-r\imunit-s\imunit\sqrt{2})(p-q\sqrt{2}+r\imunit-s\imunit\sqrt{2})(p-q\sqrt{2}-r\imunit+s\imunit\sqrt{2}) \] and compute the product using the following Python code.
\begin{minted}{python}
import sympy as sp

p, q, r, s = sp.symbols('p q r s', real=True)
sqrt2 = sp.sqrt(2)
i = sp.I

alpha = (p + q*sqrt2 + r*i + s*i*sqrt2) * (p + q*sqrt2 - r*i - s*i*sqrt2) * (p - q*sqrt2 + r*i - s*i*sqrt2) * (p - q*sqrt2 - r*i + s*i*sqrt2)

print(sp.expand(alpha))
\end{minted}
We get
\begin{align*}
    \alpha &= p^4 - 4p^2q^2 + 2p^2r^2 + 4p^2s^2 - 16pqrs + 4q^4 + 4q^2r^2 + 8q^2s^2 + r^4 - 4r^2s^2 + 4s^4 \\
    &\in S.
\end{align*}
Note that $p+q\sqrt{2}+r\imunit+s\imunit\sqrt{2} = 0$ if and only if $p=q=r=s=0$, if and only if $p+q\sqrt{2}-r\imunit-s\imunit\sqrt{2} = p-q\sqrt{2}+r\imunit-s\imunit\sqrt{2} = p-q\sqrt{2}-r\imunit+s\imunit\sqrt{2} = 0$. Hence, $\alpha = 0$ if and only if $p+q\sqrt{2}+r\imunit+s\imunit\sqrt{2} = 0$. Assume $p+q\sqrt{2}+r\imunit+s\imunit\sqrt{2} \neq 0$. Dividing the product by $\alpha$, we have \[ (p+q\sqrt{2}+r\imunit+s\imunit\sqrt{2})\lt(\frac{(p+q\sqrt{2}-r\imunit-s\imunit\sqrt{2})(p-q\sqrt{2}+r\imunit-s\imunit\sqrt{2})(p-q\sqrt{2}-r\imunit+s\imunit\sqrt{2})}{\alpha}\rt) = 1. \] So the inverse of $p+q\sqrt{2}+r\imunit+s\imunit\sqrt{2}$ is $\frac{(p+q\sqrt{2}-r\imunit-s\imunit\sqrt{2})(p-q\sqrt{2}+r\imunit-s\imunit\sqrt{2})(p-q\sqrt{2}-r\imunit+s\imunit\sqrt{2})}{\alpha}$. Since $p+q\sqrt{2}-r\imunit-s\imunit\sqrt{2}, p-q\sqrt{2}+r\imunit-s\imunit\sqrt{2}, p-q\sqrt{2}-r\imunit+s\imunit\sqrt{2}, \alpha \in S$, the inverse is also an element of $S$, proving that $S$ is a subfield. \epf

\prelem \label{prelem:ex4.2e}

The subset $S = \{p+q\sqrt{2}+r\sqrt{3}+s\sqrt{6} : p,q,r,s \in \Q\}$ of $\C$ is a subfield.

\pf Let $p_1+q_1\sqrt{2}+r_1\sqrt{3}+s_1\sqrt{6},p_2+q_2\sqrt{2}+r_2\sqrt{3}+s_2\sqrt{6} \in S$, where $p_1,q_1,r_1,s_1,p_2,q_2,r_2,s_2 \in \Q$. Note that
\begin{align*}
    & \, (p_1+q_1\sqrt{2}+r_1\sqrt{3}+s_1\sqrt{6}) + (p_2+q_2\sqrt{2}+r_2\sqrt{3}+s_2\sqrt{6}) \\
    =& \, (p_1+p_2)+(q_1+q_2)\sqrt{2}+(r_1+r_2)\sqrt{3}+(s_1+s_2)\sqrt{6} \\
    \in& \, S.
\end{align*}
Next, we can compute the product using the following Python code.
\begin{minted}{python}
import sympy as sp

p1, q1, r1, s1, p2, q2, r2, s2 = sp.symbols('p1 q1 r1 s1 p2 q2 r2 s2')
sqrt2 = sp.sqrt(2)
sqrt3 = sp.sqrt(3)
sqrt6 = sp.sqrt(6)

product = (p1 + q1*sqrt2 + r1*sqrt3 + s1*sqrt6) * (p2 + q2*sqrt2 + r2*sqrt3 + s2*sqrt6)

print(sp.collect(sp.expand(product), [sqrt2, sqrt3, sqrt6]))
\end{minted}
We get
\begin{align*}
    & \, (p_1+q_1\sqrt{2}+r_1\sqrt{3}+s_1\sqrt{6})(p_2+q_2\sqrt{2}+r_2\sqrt{3}+s_2\sqrt{6}) \\
    =& \, (p_1p_2+2q_1q_2+4r_1r_2+6s_1s_2) + (p_1q_2+p_2q_1+3r_1s_2+3r_2s_1)\sqrt{2} \\
    & \, (p_1r_2+p_2r_1+2q_1s_2+2q_2s_1)\sqrt{3} + (p_1s_2+p_2s_1+q_1r_2+q_2r_1)\sqrt{6} \\
    \in& \, S.
\end{align*}
Also, we have \[ -(p_1+q_1\sqrt{2}+r_1\sqrt{3}+s_1\sqrt{6}) = (-p_1)+(-q_1)\sqrt{2}+(-r_1)\sqrt{3}+(-s_1)\sqrt{6} \in S \] and \[ 1 = 1+0\sqrt{2}+0\sqrt{3}+0\sqrt{6} \in S. \] By the subring test, $S$ is a subring of $\C$. We next find an inverse for any non-zero element in $S$ to show that it is a subfield. Let $p+q\sqrt{2}+r\sqrt{3}+s\sqrt{6} \in S$, where $p,q,r,s \in \Q$. We first define \[ \alpha = (p+q\sqrt{2}+r\sqrt{3}+s\sqrt{6})(p+q\sqrt{2}-r\sqrt{3}-s\sqrt{6})(p-q\sqrt{2}+r\sqrt{3}-s\sqrt{6})(p-q\sqrt{2}-r\sqrt{3}+s\sqrt{6}) \] and compute the product using the following Python code.
\begin{minted}{python}
import sympy as sp

p, q, r, s = sp.symbols('p q r s')
sqrt2 = sp.sqrt(2)
sqrt3 = sp.sqrt(3)
sqrt6 = sp.sqrt(6)

product = (p + q*sqrt2 + r*sqrt3 + s*sqrt6) * (p + q*sqrt2 - r*sqrt3 - s*sqrt6) * (p - q*sqrt2 + r*sqrt3 - s*sqrt6) * (p - q*sqrt2 - r*sqrt3 + s*sqrt6)

print(sp.expand(product))
\end{minted}
We get
\begin{align*}
    \alpha &= p^4-4p^2q^2-6p^2r^2-12p^2s^2+48pqrs+4q^4-12q^2r^2-24q^2s^2+9r^4-36r^2s^2+36s^4 \\
    &\in S.
\end{align*}
By Exercise 4.4, $1$, $\sqrt{2}$, $\sqrt{3}$ and $\sqrt{6}$ are linearly independent over $\Q$. Therefore, $p+q\sqrt{2}+r\sqrt{3}+s\sqrt{6}=0$ if and only if $p=q=r=s=0$, if and only if $p+q\sqrt{2}-r\sqrt{3}-s\sqrt{6} = p-q\sqrt{2}+r\sqrt{3}-s\sqrt{6} = p-q\sqrt{2}-r\sqrt{3}+s\sqrt{6} = 0$. Hence, $\alpha=0$ if and only if $p+q\sqrt{2}+r\sqrt{3}+s\sqrt{6}=0$. Assume $p+q\sqrt{2}+r\sqrt{3}+s\sqrt{6} \neq 0$. Dividing the product by $\alpha$, we have $(p+q\sqrt{2}+r\sqrt{3}+s\sqrt{6})\lt(\frac{(p+q\sqrt{2}-r\sqrt{3}-s\sqrt{6})(p-q\sqrt{2}+r\sqrt{3}-s\sqrt{6})(p-q\sqrt{2}-r\sqrt{3}+s\sqrt{6})}{\alpha}\rt) = 1$. So the inverse of $p+q\sqrt{2}+r\sqrt{3}+s\sqrt{6}$ is $\frac{(p+q\sqrt{2}-r\sqrt{3}-s\sqrt{6})(p-q\sqrt{2}+r\sqrt{3}-s\sqrt{6})(p-q\sqrt{2}-r\sqrt{3}+s\sqrt{6})}{\alpha}$. Since $p+q\sqrt{2}-r\sqrt{3}-s\sqrt{6}, p-q\sqrt{2}+r\sqrt{3}-s\sqrt{6}, p-q\sqrt{2}-r\sqrt{3}+s\sqrt{6}, \alpha \in S$, the inverse is also an element of $S$, proving that $S$ is a subfield. \epf





\prelem \label{prelem:ex4.5b}

Let $K$ be a subfield of $\C$. Then
\begin{align*}
    K(\alpha) &= \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\} \\
    &= \lt\{ \frac{a_0+\ldots+a_n\alpha^n}{b_0+\ldots+b_m\alpha^m} : a_0,\ldots,a_n,b_0,\ldots,b_m \in K, b_0+\ldots+b_m\alpha^m \neq 0 \rt\}.
\end{align*}

\pf First, note that the second equality can be obtained by writing $p(\alpha) = a_0+\ldots+a_n\alpha^n$ and $q(\alpha) = b_0+\ldots+b_m\alpha^m$.

To prove the first equality, recall that $K(\alpha)$ is the smallest subfield of $\C$ generated by $K \cup \{\alpha\}$. Since $K \subseteq K(\alpha)$ and $\alpha \in K(\alpha)$, we have $k_0+\ldots+k_n\alpha^n \in K(\alpha)$ for any $k_0,\ldots,k_n \in K$. If $k_0+\ldots+k_n\alpha^n \neq 0$, then $\frac{1}{k_0+\ldots+k_n\alpha^n} \in K(\alpha)$. Therefore, for any $a_0,\ldots,a_n,b_0,\ldots,b_m \in K$ with $b_0+\ldots+b_m\alpha^m \neq 0$, we have \[ \frac{a_0+\ldots+a_n\alpha^n}{b_0+\ldots+b_m\alpha^m} \in K(\alpha). \] So \[ \lt\{ \frac{a_0+\ldots+a_n\alpha^n}{b_0+\ldots+b_m\alpha^m} : a_0,\ldots,a_n,b_0,\ldots,b_m \in K, b_0+\ldots+b_m\alpha^m \neq 0 \rt\} \subseteq K(\alpha). \]

Next, we want to prove that $\lt\{ \frac{a_0+\ldots+a_n\alpha^n}{b_0+\ldots+b_m\alpha^m} : a_0,\ldots,a_n,b_0,\ldots,b_m \in K, b_0+\ldots+b_m\alpha^m \neq 0 \rt\} = \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}$ is a subfield of $\C$. Let \[\frac{p(\alpha)}{q(\alpha)}, \frac{\hat{p}(\alpha)}{\hat{q}(\alpha)} \in \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\},\] where $q(\alpha), \hat{q}(\alpha) \neq 0$. Since $K$ is a field and thus a domain, we have $q(\alpha)\hat{q}(\alpha) \neq 0$. Note that
\begin{align*}
    \frac{p(\alpha)}{q(\alpha)} + \frac{\hat{p}(\alpha)}{\hat{q}(\alpha)} &= \frac{p(\alpha)\hat{q}(\alpha) + \hat{p}(\alpha)q(\alpha)}{q(\alpha)\hat{q}(\alpha)} \\
    &\in \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}, \\
    \frac{p(\alpha)}{q(\alpha)} \cdot \frac{\hat{p}(\alpha)}{\hat{q}(\alpha)} &= \frac{p(\alpha)\hat{p}(\alpha)}{q(\alpha)\hat{q}(\alpha)} \\
    &\in \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}, \\
    -\frac{p(\alpha)}{q(\alpha)} &= \frac{-p(\alpha)}{q(\alpha)} \\
    &\in \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}.
\end{align*}
Also, since $K$ is a subfield of $\C$, we have $1 \in K$. So $1 = \frac{1}{1} \in \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}$. By the subring test, $\lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}$ is a subring of $\C$.

Now, suppose $\frac{p(\alpha)}{q(\alpha)} \neq 0$. If $p(\alpha) = 0$, then $\frac{p(\alpha)}{q(\alpha)} = \frac{0}{q(\alpha)} = 0$, a contradiction. Hence, $p(\alpha) \neq 0$. So we have $\frac{q(\alpha)}{p(\alpha)} \in \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}$. Note that $\frac{p(\alpha)}{q(\alpha)} \cdot \frac{q(\alpha)}{p(\alpha)} = 1$, so the inverse of $\frac{p(\alpha)}{q(\alpha)}$ is $\frac{q(\alpha)}{p(\alpha)}$. Therefore, $\lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}$ is a subfield of $\C$. Since $\lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}$ is a field containing $K \cup \{\alpha\}$, we have $K(\alpha) \subseteq \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\}$. Hence, \[K(\alpha) = \lt\{ \frac{p(\alpha)}{q(\alpha)} : p(\alpha),q(\alpha) \in K[\alpha], q(\alpha) \neq 0 \rt\},\] proving the first equality. \epf

\ex{Exercise 1.2}

Assume for contradiction that there exists $a,b \in \Z$ with $b \neq 0$ such that $\lt(\frac{a}{b}\rt)^2=2$. If $a$ is negative and $b$ is positive, then replacing $a$ with $-a$ gives us $\lt(\frac{-a}{b}\rt)^2=\lt(\frac{a}{b}\rt)^2=2$. Therefore, $-a,b$ is also a valid pair of integers satisfying the conditions. If $b$ is negative and $a$ is positive, then replacing $b$ with $-b$ gives us $\lt(\frac{a}{-b}\rt)^2=\lt(\frac{a}{b}\rt)^2=2$. Therefore, $a,-b$ is also a valid pair of integers satisfying the conditions. If both $a$ and $b$ are negative, then replacing $a$ with $-a$ and $b$ with $-b$ gives us $\lt(\frac{-a}{-b}\rt)^2=\lt(\frac{a}{b}\rt)^2=2$. Therefore, $-a,-b$ is also a valid pair of integers satisfying the conditions. Hence, we can assume without loss of generality that $a,b>0$.

Since we assume that $b$ only takes values in $\N$, by the well-ordering principle, there exists a smallest $b$ satisfying the conditions. Note that
\begin{align*}
    \lt(\frac{2b-a}{a-b}\rt)^2 &= \frac{4b^2-4ab+a^2}{a^2-2ab+b^2} \\
    &= \frac{4-4\lt(\frac{a}{b}\rt)+\lt(\frac{a}{b}\rt)^2}{\lt(\frac{a}{b}\rt)^2-2\lt(\frac{a}{b}\rt)+1} \\
    &= \frac{4-4\lt(\frac{a}{b}\rt)+2}{2-2\lt(\frac{a}{b}\rt)+1} \\
    &= \frac{6-4\lt(\frac{a}{b}\rt)}{3-2\lt(\frac{a}{b}\rt)} \\
    &= \frac{6-4\lt(\frac{a}{b}\rt)}{3-2\lt(\frac{a}{b}\rt)} \times \frac{3+2\lt(\frac{a}{b}\rt)}{3+2\lt(\frac{a}{b}\rt)} \\
    &= \frac{18 + 12\lt(\frac{a}{b}\rt) - 12\lt(\frac{a}{b}\rt) - 8\lt(\frac{a}{b}\rt)^2}{9-4\lt(\frac{a}{b}\rt)^2} \\
    &= \frac{18-8\times2}{9-4\times2} \\
    &= 2.
\end{align*}
Recall that $\lt(\frac{a}{b}\rt)^2=2$, so we have $a^2=2b^2$ and thus $2b^2-a^2=0$. Note that
\begin{align*}
    (2b-a)(2b+a) &= 4b^2-a^2 \\
    &= 2b^2 + (2b^2-a^2) \\
    &= 2b^2 \\
    &> 0
\end{align*}
since $b>0$ by assumption. Since $a,b>0$, we have $2b+a>0$. Hence, $2b-a>0$. Also, from $a^2=2b^2$ we have $a^2-b^2=b^2$. Considering $b>0$, we have
\begin{align*}
    b^2 &> 0 \\
    a^2-b^2 &> 0 \\
    (a-b)(a+b) &> 0 \\
    a-b &> 0
\end{align*}
since $a+b>0$. From $2b-a>0$ and $a-b>0$ we know that $2b-a,a-b$ is also a pair of positive integers satisfying the conditions. However, from $2b-a>0$ we have $a<2b$, and hence $a-b<2b-b=b$, contradicting the minimality of $b$. \epf

\ex{Exercise 1.5}

Let $S = \{p+q\alpha+r\alpha^2 \mid p,q,r\in\Q\}$. We first prove that $S$ is a subring of $\C$. 

Note that $1=1+0\alpha+0\alpha^2\in S$. Let $p+q\alpha+r\alpha^2,p'+q'\alpha+r'\alpha^2 \in S$, where $p,q,r,p',q',r' \in \Q$. Then we have $(p+q\alpha+r\alpha^2)+(p'+q'\alpha+r'\alpha^2) = (p+p') + (q+q')\alpha + (r+r')\alpha^2 \in S$, $-(p+q\alpha+r\alpha^2) = (-p) + (-q)\alpha + (-r)\alpha^2 \in S$ and $(p+q\alpha+r\alpha^2)(p'+q'\alpha+r'\alpha^2) = (pp'+2rq') + (pq'+p'q+2rr')\alpha + (pr'+qq'+qr'+rp')\alpha^2 \in S$. By the subring test, $S$ is a ring.

Next, we construct an inverse explicitly to show that $S$ is a subfield. Let $x = p+q\alpha+r\alpha^2 \in S \setminus \{0\}$, where $p,q,r \in \Q$. Then we have $x^2=(p^2+4qr)+(2pq+2r^2)\alpha+(2pr+q^2)\alpha^2$ and $x^3=(p^3+12pqr+2q^3+4r^3)+(3p^2q+6pr^2+6q^2r)\alpha+(3p^2r+3pq^2+6qr^2)\alpha^2$. Let $x^3+k_2x^2+k_1x+k_0=0$. Denote the coefficient of $\alpha^j$ in $x^i$ by $x_{ij}$. Then we have $(x_{30}+x_{31}\alpha+x_{32}\alpha^2)+k_2(x_{20}+x_{21}\alpha+x_{22}\alpha^2)+k_1(p+q\alpha+r\alpha^2)+k_0=0$, that is, $(x_{30}+x_{20}k_2+pk_1+k_0) + (x_{31}+x_{21}k_2+qk_1)\alpha + (x_{32}+x_{22}k_2+rk_1)\alpha^2 = 0$. Since $1,\alpha,\alpha^2$ are linearly independent, comparing coefficients gives us the linear system
\[ \begin{cases}
    k_0 + pk_1 + x_{20}k_2 = -x_{30} \\
    qk_1 + x_{21}k_2 = -x_{31} \\
    rk_1 + x_{22}k_2 = -x_{32}
\end{cases}. \]
We can use the following python code to solve this system.

\begin{minted}{python}
import sympy as sp

k0, k1, k2 = sp.symbols('k0 k1 k2')
p, q, r = sp.symbols('p q r')

x30 = p**3 + 12 * p * q * r + 2 * (q**3) + 4 * (r**3)
x31 = 3 * (p**2) * q + 6 * p * (r**2) + 6 * (q**2) * r
x32 = 3 * (p**2) * r + 3 * p * (q**2) + 6 * q * (r**2)
x20 = p**2 + 4 * q * r
x21 = 2 * p * q + 2 * (r**2)
x22 = 2 * p * r + q**2

M = sp.Matrix([
    [1, p, x20, - x30],
    [0, q, x21, - x31],
    [0, r, x22, - x32]
])

solution = sp.solve_linear_system(M, k0, k1, k2)

for var in [k0, k1, k2]:
    print(f"{var} =")
    sp.pprint(sp.simplify(solution[var]))
\end{minted}

The solution is $k_0 = -p^3+6pqr-2q^3-4r^3$, $k_1=3p^2-6qr$ and $k_2=-3p$. Since $x^3+k_2x^2+k_1x+k_0 = 0$, we have $x(x^2+k_2x+k_1+k_0x^{-1}) = 0$. Recall that $\C$ is a field and hence a domain, and $x \neq 0$. Therefore, $x^2+k_2x+k_1+k_0x^{-1} = 0$, that is,
\[x^{-1} = -\frac{1}{k_0}x^2 -\frac{k_2}{k_0}x -\frac{k_1}{k_0}.\]
Since $x$ and $x^2$ are linear combinations of $1,\alpha,\alpha^2$, $x^{-1}$ is also a linear combination of $1,\alpha,\alpha^2$. \epf

    1 & 1 & 1 & 1 & \cdots & 1 \\
    1 & 2 & 2^2 & 2^3 & \cdots & 2^n \\

\ex{Exercise 4.1}

For field extensions $\iota$ and $j$, define $\iota \sim j$ as $\iota$ is isomorphic to $j$. Let $\iota_1 : K_1 \to \hat{K}_1$, $\iota_2 : K_2 \to \hat{K}_2$ and $\iota_3 : K_3 \to \hat{K}_3$ be field extensions.

For the reflexive property, we want to prove that there exists field isomorphisms $\lambda : K_1 \to K_1$ and $\mu : \hat{K}_1 \to \hat{K}_1$, such that \[ \iota_1(\lambda(k)) = \mu(\iota_1(k)) \] for all $k \in K_1$. Consider $\lambda$ and $\mu$ defined by $\lambda(k)=k$ and $\mu(\hat{k})=\hat{k}$ for all $k \in K_1$ and $\hat{k} \in \hat{K}$. Clearly, $\lambda$ and $\mu$ are field isomorphisms. Also, for any $k \in K_1$, we have \[ \iota_1(\lambda(k)) = \iota_1(k) = \mu(\iota_1(k)). \] Therefore, $\iota_1 \sim \iota_1$.

For the symmetric property, assume $\iota_1 \sim \iota_2$, so there exist field isomorphisms $\lambda_1 : K_1 \to K_2$ and $\mu : \hat{K}_1 \to \hat{K}_2$ such that $\iota_2(\lambda_1(k))=\mu_1(\iota_1(k))$ for all $k \in K_1$. We want to prove that $\iota_2 \sim \iota_1$, that is, there exist field isomorphisms $\lambda_2 : K_2 \to K_1$ and $\mu_2 : \hat{K}_2 \to \hat{K}_1$ such that $\iota_1(\lambda_2(k)) = \mu_2(\iota_2(k))$ for all $k \in K_2$. Since $\lambda_1$ and $\mu_1$ are bijective, they have inverses $\lambda_1^{-1}$ and $\mu_1^{-1}$. Let $\lambda_2=\lambda_1^{-1}$ and $\mu_2=\mu_1^{-1}$. Then
\begin{align*}
    \iota_1(\lambda_2(k)) &= \iota_1\lt(\lambda_1^{-1}(k)\rt) \\
    &= \mu_1^{-1}\lt(\mu_1\lt(\iota_1\lt(\lambda_1^{-1}(k)\rt)\rt)\rt) \\
    &= \mu_2\lt(\iota_2\lt(\lambda_1\lt(\lambda_1^{-1}(k)\rt)\rt)\rt) \\
    &= \mu_2(\iota_2(k))
\end{align*}
for any $k \in K_2$. Therefore, $\iota_2 \sim \iota_1$.

For the transitive property, suppose $\iota_1 \sim \iota_2$ and $\iota_2 \sim \iota_3$, so there exist field isomorphisms $\lambda_1 : K_1 \to K_2$, $\mu_1 : \hat{K}_1 \to \hat{K}_2$, $\lambda_2 : K_2 \to K_3$ and $\mu_2 : \hat{K}_2 \to \hat{K}_3$ such that $\iota_2(\lambda_1(k))=\mu_1(\iota_1(k))$ for all $k \in K_1$ and $\iota_3(\lambda_2(k))=\mu_2(\iota_2(k))$ for all $k \in K_2$. We want to prove that $\iota_1 \sim \iota_3$, that is, there exist field isomorphisms $\lambda_3 : K_1 \to K_3$ and $\mu_3 : \hat{K}_1 \to \hat{K}_3$ such that $\iota_3(\lambda_3(k))=\mu_3(\iota_1(k))$ for all $k \in K_1$. Consider $\lambda_3 = \lambda_2 \circ \lambda_1$ and $\mu_3 = \mu_2 \circ \mu_1$. Note that $\lambda_3$ and $\mu_3$ are field isomorphisms because the composition of field isomorphisms is also a field isomorphism. We have
\begin{align*}
    \iota_3(\lambda_3(k)) &= \iota_3(\lambda_2(\lambda_1(k))) \\
    &= \mu_2(\iota_2(\lambda_1(k))) \\
    &= \mu_2(\mu_1(\iota_1(k))) \\
    &= \mu_3(\iota_1(k))
\end{align*}
for any $k \in K_1$. Therefore, $\iota_1 \sim \iota_3$.

Hence, the isomorphism of field extensions is an equivalence relation. \epf
